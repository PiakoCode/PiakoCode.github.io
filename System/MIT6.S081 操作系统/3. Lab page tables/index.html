    и§ЈйҮҠиҝҮзЁӢеӣҫпјҡ

    1. зЁӢеәҸиҝҗиЎҢж—¶пјҢзЁӢеәҸдҪҝз”ЁиҷҡжӢҹең°еқҖи®ҝй—®еҶ…еӯҳгҖӮ
    2. MMUпјҲMemory Management UnitпјүзЎ¬д»¶жЁЎеқ—жҳҜи®Ўз®—жңәзҡ„дёҖйғЁеҲҶпјҢиҙҹиҙЈиҷҡжӢҹеҶ…еӯҳең°еқҖиҪ¬жҚўгҖӮе®ғжҺҘж”¶зЁӢеәҸеҸ‘еҮәзҡ„иҷҡжӢҹең°еқҖпјҢе№¶йҖҡиҝҮйЎөиЎЁжҳ е°„е°ҶиҷҡжӢҹең°еқҖиҪ¬жҚўдёәзү©зҗҶең°еқҖгҖӮ
    3. ж“ҚдҪңзі»з»ҹзҡ„йЎөиЎЁжҳҜеӯҳеӮЁеңЁеҶ…еӯҳдёӯзҡ„ж•°жҚ®з»“жһ„пјҢе®ғеҢ…еҗ«иҷҡжӢҹйЎөеҸ·дёҺзү©зҗҶйЎөеҸ·д№Ӣй—ҙзҡ„жҳ е°„е…ізі»гҖӮMMUдјҡдҪҝз”ЁиҷҡжӢҹйЎөеҸ·еңЁйЎөиЎЁдёӯжҹҘжүҫеҜ№еә”зҡ„зү©зҗҶйЎөеҸ·гҖӮ
    4. зү©зҗҶеҶ…еӯҳжҳҜе®һйҷ…зҡ„еҶ…еӯҳз©әй—ҙпјҢеҢ…еҗ«дәҶзЁӢеәҸзҡ„ж•°жҚ®е’ҢжҢҮд»ӨгҖӮдёҖж—ҰжүҫеҲ°иҷҡжӢҹйЎөеҸ·еҜ№еә”зҡ„зү©зҗҶйЎөеҸ·пјҢMMUдјҡе°Ҷзү©зҗҶйЎөеҸ·дёҺйЎөеҶ…еҒҸз§»з»„еҗҲжҲҗзү©зҗҶең°еқҖпјҢд»ҺиҖҢиғҪеӨҹи®ҝй—®е®һйҷ…зҡ„зү©зҗҶеҶ…еӯҳгҖӮ

    йҖҡиҝҮиҝҷдёӘиҝҮзЁӢпјҢзЁӢеәҸиғҪеӨҹеңЁиҷҡжӢҹеҶ…еӯҳз©әй—ҙдёӯиҝҗиЎҢпјҢе№¶дё”дёҚйңҖиҰҒе…іеҝғзү©зҗҶеҶ…еӯҳзҡ„е…·дҪ“дҪҚзҪ®е’Ңз®ЎзҗҶпјҢж“ҚдҪңзі»з»ҹйҖҡиҝҮиҷҡжӢҹеҶ…еӯҳең°еқҖиҪ¬жҚўжқҘе®ҢжҲҗеҶ…еӯҳз®ЎзҗҶзҡ„д»»еҠЎгҖӮ

    еӨҡзә§йЎөиЎЁ-еҗҚиҜҚ¶

    йЎөпјҲPageпјү¶

    йЎөпјҲйҖҡеёёз§°дёәйЎөйқўпјүжҳҜеҶ…еӯҳз®ЎзҗҶдёӯзҡ„дёҖдёӘеҹәзЎҖжҰӮеҝөпјҢе®ғд»ЈиЎЁзҡ„жҳҜдёҖдёӘеӣәе®ҡй•ҝеәҰзҡ„еҶ…еӯҳеқ—гҖӮж“ҚдҪңзі»з»ҹжҠҠзү©зҗҶеҶ…еӯҳе’ҢиҷҡжӢҹеҶ…еӯҳйғҪеҲҶдёәи®ёеӨҡеӨ§е°Ҹзӣёзӯүзҡ„йЎөгҖӮдҫӢеҰӮпјҢеңЁеҫҲеӨҡзі»з»ҹдёӯпјҢдёҖдёӘеёёи§Ғзҡ„йЎөеӨ§е°ҸеҸҜиғҪжҳҜ4KBгҖӮдҪҝз”ЁйЎөзҡ„дё»иҰҒзӣ®зҡ„жҳҜдёәдәҶе®һзҺ°иҷҡжӢҹеҶ…еӯҳзі»з»ҹпјҢе…¶дёӯзҡ„ең°еқҖз©әй—ҙз”ұи®ёеӨҡиҝҷж ·зҡ„йЎөжһ„жҲҗгҖӮ

    йЎөиЎЁпјҲPage Tableпјү¶

    йЎөиЎЁжҳҜдёҖз§Қж•°жҚ®з»“жһ„пјҢе®ғжҠҠиҷҡжӢҹең°еқҖз©әй—ҙдёӯзҡ„йЎөжҳ е°„еҲ°зү©зҗҶеҶ…еӯҳдёӯзҡ„йЎөеё§гҖӮиҝҷж ·зҡ„жҳ е°„еҜ№дәҺж“ҚдҪңзі»з»ҹиҝӣиЎҢең°еқҖиҪ¬жҚўжҳҜеҝ…иҰҒзҡ„вҖ”еҚіе°ҶзЁӢеәҸдҪҝз”Ёзҡ„иҷҡжӢҹең°еқҖиҪ¬жҚўдёәе®һйҷ…зү©зҗҶеҶ…еӯҳең°еқҖгҖӮйЎөиЎЁйҖҡеёёеӯҳеӮЁеңЁеҶ…еӯҳдёӯпјҢжңүж—¶еҖҷиҝҳдјҡиў«зј“еӯҳеҲ°еҝ«йҖҹзҡ„зЎ¬д»¶еҜ„еӯҳеҷЁпјҲдҫӢеҰӮTLBвҖ”Translation Lookaside Bufferпјүдёӯд»ҘеҠ йҖҹжҹҘжүҫйҖҹеәҰгҖӮ

    йЎөзӣ®еҪ•пјҲPage Directoryпјү¶

    еңЁи®ёеӨҡеҲҶйЎөзі»з»ҹдёӯпјҢзү№еҲ«жҳҜеңЁIntel x86дҪ“зі»з»“жһ„зҡ„дј з»ҹеҲҶйЎөжЁЎејҸдёӯпјҢйЎөзӣ®еҪ•жҳҜз®ЎзҗҶйЎөиЎЁзҡ„дёҖз§ҚжңәеҲ¶гҖӮиҖғиҷ‘еҲ°йЎөиЎЁеҸҜиғҪдјҡйқһеёёеӨ§пјҢдёҖдёӘеҢ…еҗ«жүҖжңүжҳ е°„зҡ„еҚ•дёҖйЎөиЎЁе°ҶдёҚе®һз”ЁпјҢжүҖд»Ҙзі»з»ҹдҪҝз”ЁйЎөзӣ®еҪ•жқҘз»„з»ҮиҝҷдәӣйЎөиЎЁгҖӮйЎөзӣ®еҪ•еҢ…еҗ«дәҶжҢҮеҗ‘йЎөиЎЁзҡ„жҢҮй’ҲпјҢжҜҸдёӘжҢҮй’ҲйғҪеҜ№еә”зқҖдёҖдёӘйЎөиЎЁпјҢиҝҷдәӣйЎөиЎЁиҝӣдёҖжӯҘеӯҳеӮЁзқҖиҷҡжӢҹйЎөеҲ°зү©зҗҶйЎөеё§зҡ„жҳ е°„гҖӮйЎөзӣ®еҪ•жң¬иә«д№ҹеҲҶйЎөпјҢе°Ҷең°еқҖз©әй—ҙзҡ„дёҚеҗҢйғЁеҲҶеҲҶй…Қз»ҷдёҚеҗҢзҡ„йЎөиЎЁгҖӮ

    йЎөиЎЁйЎ№пјҲPage Table EntryпјҢPTEпјү¶

    жҜҸдёӘйЎөиЎЁйЎ№жҳҜйЎөиЎЁдёӯзҡ„дёҖдёӘе…ғзҙ пјҢеҢ…еҗ«з”ЁдәҺе°ҶиҷҡжӢҹйЎөжҳ е°„еҲ°зү©зҗҶйЎөеё§зҡ„дҝЎжҒҜгҖӮPTEйҖҡеёёеҢ…еҗ«зү©зҗҶйЎөеё§зҡ„ең°еқҖд»ҘеҸҠдёҺиҜҘйЎөзӣёе…ізҡ„дёҖдәӣж Үеҝ—пјҢеҰӮжҳҜеҗҰеҸҜиҜ»еҶҷпјҢжҳҜеҗҰе·Іиў«и®ҝй—®пјҢжҳҜеҗҰи„ҸпјҲиў«дҝ®ж”№иҝҮпјүпјҢд»ҘеҸҠйЎөжҳҜеҗҰеңЁзү©зҗҶеҶ…еӯҳдёӯгҖӮ

    йЎөйқў¶

    йЎөйқўдёҺйЎөжҳҜзӣёеҗҢзҡ„жҰӮеҝөгҖӮеңЁеҲҶйЎөеҶ…еӯҳз®ЎзҗҶдёӯпјҢвҖңйЎөйқўвҖқжҳҜеҸҰдёҖз§ҚиҜҙжі•пјҢжҢҮзҡ„жҳҜиў«еҲҶй…ҚдәҶзү№е®ҡзү©зҗҶеҶ…еӯҳйЎөеё§зҡ„иҷҡжӢҹеҶ…еӯҳеҢәеҹҹгҖӮ

    йЎөжЎҶеҸ·=йЎөеё§еҸ·=еҶ…еӯҳеқ—еҸ·=зү©зҗҶеқ—еҸ·=зү©зҗҶйЎөеҸ·

    д»Һеұһе…ізі»¶

    1. йЎөйқўпјҲPageпјүпјҡ

    void vmprint(pagetable_t pagetable,int depth) {
  if(depth > 2) { // ж·ұеәҰеӨ§дәҺ2е°ұиҝ”еӣһ
    return;
  }
  if(depth == 0) { // ж·ұеәҰзӯүдәҺ0ж—¶иҫ“еҮәpagetable
    printf("page table %p\n",pagetable);
  }

  for(int i = 0; i < 512; i++){ // йҒҚеҺҶйЎөиЎЁйЎ№
    pte_t pte = pagetable[i];
    if(pte & PTE_V) { // еҰӮжһңеҸҜд»Ҙи®ҝй—®

      for(int i=0;i<=depth;i++){ // еӨ„зҗҶvmprintзҡ„ж јејҸ
        if(i == 0)
          printf("..");
        else {
          printf(" ..");
        }
      }

      printf("%d: pte %p pa %p\n",i,pte,PTE2PA(pte)); // иҫ“еҮә

      uint64 child = PTE2PA(pte); // дёӢдёҖзә§
      vmprint((pagetable_t)child,depth+1);
    }
  }
}
