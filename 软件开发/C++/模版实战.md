# 模版实战

#C++

```cpp
#include <iostream>

using namespace std;

template <typename T>
void my_swap(T &a, T &b) // 非优先调用
{
    auto temp = a;
    a = b;
    b = temp;
}
template <typename T, typename U>
void my_swap(T &a, U &b)
{
    auto temp = a;
    a = b;
    b = temp;
}

void my_swap(int &a, int &b) // 优先调用
{
    cout << "YYY" << endl;
    auto temp = a;
    a = b;
    b = temp;
}

int main()
{
    int aa = 1;
    int bb = 2;
    my_swap(aa, bb); // 调用int
    cout << aa << " " << bb << endl;

    double a = 1;
    float b = 2.1;
    my_swap(a, b);
    cout << a << " " << b << endl;
    return 0;
}
```

```
YYY
21
2.1 1
```

模版函数不支持隐式转换，必须匹配
当函数重载与模版函数冲突时，优先调用普通函数
